## Terraform CLI 실습 

- `terraform fmt`: 자동 포맷팅 (현재 작업중인 디렉토리만)
  - 하위 디렉터리 까지 포맷팅 하려면 `-recursive` 옵션 추가하기

```hcl
provider "random" {

}

resource "random_pet" "name" {
  length = 2
}

output "random_pet_name" {
  value = random_pet.name.id
}
```

### 환경 변수 사용하기  

```hcl
provider "random" {

}

variable "num_of_pets" {
  type        = number
  description = "how many pets do we want"
}


resource "random_pet" "name" {
  length = var.num_of_pets
}


output "random_pet_name" {
  value = random_pet.name.id
}
```

- `export TF_VAR_num_of_pets = 5`로 설정 후 terraform plan 실행

- 환경 변수 임시 설정: export 사용 (터미널 끄면 사라진다)
- 영구 설정 (.zshrc 파일 수정)
- 변수 우선순위
  - `TF_VAR_...` (가장 낮음)
  - `terraform.tfvars` 파일
  - `*.auto.tfvars` 파일
  - 명령어 옵션 (-var or -var-file)
- `TF_LOG`: 테라폼이 실행될 때 나오는 로그의 상세도를 조절한다. (DEBUG, INFO 등) 공부하거나 에러를 잡을 때 사용한다.
