# OCI Free tier 가입 이유 및 제공해주는 것들

- 소모임 클라우드 환경 실습을 위해 Free tier 임에도 좋은 성능을 자랑하는 컴퓨트를 제공하는 OCI에 가입했다.


| 분류       | 서비스 및 스펙                                                                                  |
|------------|-------------------------------------------------------------------------------------------|
| 컴퓨트     | AMD 기반 VM: 2개 (각 1/8 OCPU, 1GB RAM)<br>ARM 기반 Ampere A1: 4 OCPU, 24GB RAM (VM 1~4개로 분할 가능) |
| 스토리지   | 블록 볼륨: 총 200GB (2개)<br>오브젝트 스토리지: 10GB (표준), 10GB (비정규), 10GB (아카이브)                      |
| 데이터베이스 | Autonomous Database: 2개 (각 1 OCPU, 20GB)<br>NoSQL Database: 월 1.33억 읽기/쓰기, 테이블 3개 (각 25GB) |
| 네트워크   | 아웃바운드 트래픽: 월 10TB<br>로드 밸런서: 1개 (10Mbps)<br>VCN: 2개 <br>고정 IP 1개 제공                       |


# Oracle Cloud Free Tier 인스턴스 사양

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인스턴스 이름 | monithub-Instance |
| 리전 | US East (Ashburn) |
| 가용성 도메인 | AD-3 |
| 용량 유형 | On-Demand (Free Tier) |

## 컴퓨팅 사양

| 항목 | 내용 |
|------|------|
| Shape | VM.Standard.A1.Flex |
| 아키텍처 | ARM (Ampere) |
| OCPU | 4코어 |
| 메모리 | 24GB |
| 네트워크 대역폭 | 4Gbps |

## OS

| 항목 | 내용 |
|------|------|
| 운영체제 | Canonical Ubuntu 22.04 Minimal aarch64 |
| 이미지 빌드 | 2026.02.28-0 |

## 스토리지

| 항목 | 내용 |
|------|------|
| 부트 볼륨 크기 | 200GB |
| 부트 볼륨 성능 | VPU 10 (균형 조정) |
| 전송 암호화 | 사용 |

## 네트워크

| 항목 | 내용 |
|------|------|
| VCN | vcn-20260423-0113 |
| 서브넷 | subnet-20260423-0113 |
| CIDR 블록 | 10.0.0.0/24 |
| 프라이빗 IP | 10.0.0.10 |
| 퍼블릭 IP | Reserved (monithub-instance-ip) |

## 접속 방법

```bash
ssh -i <프라이빗_키_파일경로> ubuntu@<퍼블릭_IP>
```

## 참고사항

- Free Tier ARM 한도: OCPU 4코어, 메모리 24GB, 스토리지 200GB (현재 전부 사용)
- Reserved IP 1개 무료 제공
- 유휴 인스턴스 회수 주의: CPU 사용률이 장기간 낮으면 Oracle이 인스턴스 회수할 수 있음
